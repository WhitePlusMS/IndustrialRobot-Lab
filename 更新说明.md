## 2025-06-18 修复关节滑块拖动延迟

- 问题：拖动滑块时每次 `onChange` 都直接调用 `onAdjustJoint`，事件风暴导致 App + 3D 场景逐帧重绘，产生明显延迟
- 解决：`JointAngleCard` 内通过 `requestAnimationFrame` 节流，把高频 `onChange` 合并为每帧一次提交
- 拖动过程中机器人实时跟随（按显示器刷新率更新），同时避免事件风暴
- `npm run check` 通过，无 TypeScript 错误

## 2025-06-18 优化关节滑块跟踪流畅度

- 问题：滑块 thumb 受控于父组件 `joints`，要等 React 渲染完才能移动，导致跟不上鼠标
- 解决：增加本地 `sliderValues` 状态，拖动时立即更新 thumb 和显示数值；父组件仍按 `requestAnimationFrame` 节流更新
- 松开鼠标时 flush 最后一帧 pending 更新并清掉本地覆盖，避免后续外部更新被覆盖
- `npm run check` 通过，无 TypeScript 错误
